Automatically replaces Contact Form 7 file upload fields with clickable download URLs in email notifications.

== Description ==

CF7 File URL Replacer enhances Contact Form 7 by replacing file field tags with clickable download URLs in email notifications. Files are permanently saved to your WordPress Media Library.

= Features =

* Automatically saves uploaded files to WordPress Media Library
* Replaces all [file-*] tags with clickable download links
* Supports multiple file uploads
* Professional email formatting with file size display
* Works with all Contact Form 7 file fields
* No configuration required - works out of the box
* Lightweight and performance optimized

= How It Works =

1. Install and activate the plugin
2. Create your Contact Form 7 form with file upload fields
3. Use file field tags like [file-123] in your email template
4. Files are automatically saved to Media Library
5. Email recipients receive clickable download links

= Requirements =

* Contact Form 7 plugin must be installed and activated
* Enable "Use HTML content type" in CF7 mail settings for styled links

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/cf7-file-url-replacer` directory, or install through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Make sure Contact Form 7 is installed and activated
4. Enable "Use HTML content type" in your Contact Form 7 mail settings
5. Use file field tags in your email templates - they'll be automatically replaced

== Frequently Asked Questions ==

= Does this work with Contact Form 7? =

Yes, Contact Form 7 must be installed and activated for this plugin to work.

= Do I need to configure anything? =

No configuration needed. Just activate and it works automatically with all CF7 forms.

= Where are the files stored? =

Files are permanently saved to your WordPress Media Library.

= Can I use multiple file upload fields? =

Yes, it works with unlimited file upload fields in your forms.

= Will this work with existing forms? =

Yes, it works with all existing and new Contact Form 7 forms.
